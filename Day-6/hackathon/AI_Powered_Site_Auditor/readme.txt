🧠 AI-Powered Website Compliance Auditor

A smart website auditor that automatically checks any URL for compliance with key regulations including GDPR, Accessibility (ADA/WCAG), SEO, and Security.

Built for speed and clarity — it scrapes a site, analyzes its structure, evaluates compliance, and uses AI and RAG to generate actionable insights and reports.

🚀 Features

* ✅ Website scraping and content analysis
* 🛡️ GDPR, WCAG/ADA, SEO & Security compliance checks
* 🔍 AI-powered insights using **Gemini** (Google Generative AI)
* 📚 RAG (Retrieval-Augmented Generation) system with **ChromaDB**
* 📘 Regulatory guidance pulled from official sources (GDPR, WCAG, ADA)
* 📈 Visual compliance reports via **Gradio** interface

🧩 Tech Stack

* Python, Gradio
* Gemini (Google Generative AI)
* ChromaDB (Vector DB for RAG)
* BeautifulSoup, Selenium
* Pillow, Contrast-Ratio
* LangChain – for modular prompt chaining and retrieval handling
